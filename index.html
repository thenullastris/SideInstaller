រៀបចំដំបូង

ដំឡើងប្រវត្តិរូប DNS របស់ SideHelper មុនអ្វីទាំងអស់។

SideHelper ត្រូវការឲ្យ DNS របស់វាដំណើរការនៅលើឧបករណ៍អ្នកកំឡុងពេលរៀបចំ។ ចុចប៊ូតុងខាងក្រោម រួចអនុវត្តតាមជំហានដើម្បីបើកវា៖

ដំឡើងប្រវត្តិរូប DNS
  1. ចុច ដំឡើងប្រវត្តិរូប DNS ខាងលើ រួចចុច Allow នៅពេល Safari សួរទាញយកឯកសារកំណត់រចនាសម្ព័ន្ធ។ ប្រសិនបើវាត្រូវបានរក្សាទុកជាឯកសារធម្មតា សូមបើកវាពី Files → Downloads។
  2. បើក Settings រួចចុច Profile Downloaded (ឬចូលទៅ Settings → General → VPN & Device Management) បន្ទាប់មកចុច Install។
  3. អនុវត្ត DNS៖ នៅក្នុង Settings → General → VPN & Device Management បើក DNS ហើយប្រាកដថា SideHelper ត្រូវបានជ្រើសរើស។ ទុកវាឲ្យបើករហូតដល់ការរៀបចំបានចប់សព្វគ្រប់។

របៀបដំឡើង
  1. បើកទំព័រនេះ នៅលើ iPhone ឬ iPad របស់អ្នកតាមរយៈ Safari។
  2. ជ្រើសរើសវិញ្ញាបនបត្រណាមួយខាងលើ ចុច Install រួចបញ្ជាក់ប្រអប់សួរ។
  3. រង់ចាំកម្មវិធីដំឡើងចប់នៅលើអេក្រង់ដើមរបស់អ្នក។
  4. ចូលទៅ Settings → General → VPN & Device Management ចុចប្រវត្តិរូបអ្នកអភិវឌ្ឍន៍ រួចចុច Trust។
  5. បើកកម្មវិធី។ ប្រសិនបើវិញ្ញាបនបត្រមួយត្រូវបានដកហូត ឬមិនអាចបើកបាន គ្រាន់តែដំឡើងជាមួយវិញ្ញាបនបត្រផ្សេងទៀត។

បញ្ចប់

អ្នករួចរាល់ហើយ។

រួចហើយ — បើក SideHelper ពីអេក្រង់ដើមរបស់អ្នក ហើយប្រើវាដើម្បីដំឡើង SideStore (ឬ SideStore + LiveContainer) ដោយផ្ទាល់ពីឧបករណ៍អ្នក។ មិនត្រូវការកុំព្យូទ័រទេ។

នៅពេល SideStore បានដំឡើងរួច លែងត្រូវការប្រវត្តិរូប DNS ទៀតទេ។ លុបវាចេញនៅក្នុង Settings → General → VPN & Device Management → SideHelper → Remove Profile — ឬទុកវាទុកចោល ហើយគ្រាន់តែឈប់ប្រើ DNS របស់វា ដោយប្តូរ DNS មកជា Automatic វិញនៅក្នុងម៉ឺនុយដដែល។
