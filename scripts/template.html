Setup

Install the SideInstaller DNS profile before anything else.

SideInstaller needs its DNS active on your device during setup. Tap the button, then follow the steps to apply it:

Install DNS Profile
  1. Tap Install DNS Profile above and tap Allow when Safari asks to download the configuration profile. If it gets saved as a regular file instead, open it from Files → Downloads.
  2. Open Settings and tap Profile Downloaded (or go to Settings → General → VPN & Device Management), then tap Install.
  3. Apply the DNS: in Settings → General → VPN & Device Management, open DNS and make sure SideInstaller is selected. Keep it enabled until setup is complete.

How to install
  1. Open this page on your iPhone or iPad in Safari.
  2. Pick any certificate above and tap Install, then confirm the prompt.
  3. Wait for the app to finish installing on your Home Screen.
  4. Go to Settings → General → VPN & Device Management, tap the developer profile, and Trust it.
  5. Launch the app. If one certificate is revoked or won't open, just install with another.
3

Completion

You're all set.

That's it — open SideInstaller from your Home Screen and use it to install SideStore (or SideStore + LiveContainer), straight from your device. No PC needed.

Once SideStore is installed, the DNS profile is no longer needed. Remove it in Settings → General → VPN & Device Management → SideInstaller → Remove Profile — or keep it and just stop using its DNS by switching DNS back to Automatic in the same menu.
